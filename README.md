# Statistical-Insights-into-Walkability-and-Connectivity-Across-U.S.-Cities
Built regression, ANOVA, and logistic models in R on 300+ U.S. cities to identify drivers of urban walkability. Model explained 90% of Walk Score variation (Adj. R² = 0.90). Logistic regression classified high-walkability cities with 91% accuracy (AUC = 0.96). That's 293 characters.
